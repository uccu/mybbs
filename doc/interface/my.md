# 字典


#### **用户信息字典**

**nickname**  `用户标识`

**uid**         `用户UID`

**type**        `用户的类型，-1未选择，0用户，1运维，2专家`

**label**       `标签`

**thumb**       `头像`

**sex**         `性别`

```
{
    "uid":"1",
    "nickname": "用户_1475134153",
    "type": "0",
    "label": "",
    "thumb": "x.jpg",
    "sex": "0"
}

```
